Terminal
        ========
        - built in application that provides a command line interface to interact with the computer
            - mac terminal
            - windows powershell, wsl
                - //wsl$
                - remote wsl for vscode
        - enter commands 
        - previously a physical device
        
        Shell
        =====
        - program that processes commands and returns an output
            - bash
            - zshell

        Root
        ====
        - all files and folders begin at the root directory
        - / is the root directory
        - ~ is the home directory
            - /home/username
