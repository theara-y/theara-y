        What is hashing?
        ================
        - hash function maps data of arbitrary size to fixed size
        - determininstic - same output for the same input everytime
        - cryptographic hash - one way function, cannot reverse output to get input
            - abs function is one way

        Python hashing
        ==============
        - built in function called hash
            - cannot hash mutable data types
            - dictionaries and sets use a hashing function for keys

        Salting
        =======
        - Prevents reverse lookup of hash
        - Salt is a random string introduced during hashing
        - Attackers need to guess salt + password
        - Concatenated to the password and hashed together
        - Two people with the same password will have differnt hashes



        Algorithms
        ==========
        - Cryptographic Hashes
            - MD5
            - SHA
            - Fast, non reversible
        - Password hashing
            - Argon2
            - Bcrypt
            - Scrypt
            - Slower
